# 📚 Projeto Escola API

API desenvolvida com Django Rest Framework para gerenciamento de estudantes e cursos.

## 🚀 Tecnologias
- Python
- Django
- Django REST Framework

## ⚙️ Como rodar

```bash
git clone https://github.com/kennedy-silva/escola_DRF.git
cd escola
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python manage.py runserver
